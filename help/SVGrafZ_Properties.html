SVGrafZ Properties

Tipps:

Values
  • StyleSheet:
    • The value can be a relative path. If the stylesheet is not found in the place the path directes to, the stylesheet gets aquired.
    • The stylesheet should look like this: 
      .dataset0 {fill: blanchedalmond; stroke:blanchedalmond;} 
.dataset1 {fill: lightgreen; stroke:lightgreen;}
      • The number after dataset is the number of the graph. You need at least as many dataset-entries as you have datasets.
      • The colors of the attributes fill and stroke should always be the same. Otherwise the legend will have different colors than the curves.
      • The fill attribute must not be set for LineGraphs otherwise the curves will get filled.
    • If no StyleSheet is selected or no CSS-definition if found for a dataset. Then the default color is used. It is configurable in config.py and defaultes to lightgreen.
  • Legend (TALES):
    • Use string:converter to get the legend data from the DataConverter. If the choosen DataConverter is not able to generate the legend, you will get an error message on your generated diagram.
Inheritence
  • To reset the value of an inherited TALES-Expression (e.g. ColumnNames or Legend) to no value use string: as TALES-Expression.
  • To remove the display of an inherited title attribute use a space as new title.
  • You can't reset the value of the stylesheet-path, instead create an empty file for the stylesheet.
